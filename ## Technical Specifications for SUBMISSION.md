## Technical Specifications### 1. Backend Architecture (The Engine)Language: Python 3.10+Framework: Flask (Micro-web framework)Logic Model: Stochastic Mutation Algorithm.The engine calculates oxidative stress impact specifically on Guanine (G) bases due to their low ionization potential ($~7.75 eV$).Simulates the transition from G $\rightarrow$ 8-oxoG based on a linear-no-threshold (LNT) radiation model.Environment: Virtualized via venv to manage dependencies.### 2. Frontend & UI/UX (The Dashboard)Styling: SCSS (Sassy CSS) using a modular BEM (Block Element Modifier) naming convention for scalability.Interactivity: Vanilla JavaScript (ES6) for asynchronous UI updates without page reloads.Data Visualization: Chart.js integration via CDN.Renders real-time DNA Integrity trends using a dynamic line and bar chart system.Responsive Design: Mobile-first approach using Flexbox and CSS Grid to ensure NGO workers can use the tool on tablets or phones in the field.### 3. Internationalization (i18n)System: Custom JSON-driven localization engine.Implementation: All UI strings are decoupled from the HTML logic, allowing for a seamless toggle between English and Spanish without redundant code.### 4. Deployment & DevOpsVersion Control: Git / GitHubCI/CD: Configured for auto-deployment on Render or Vercel.Documentation: Comprehensive Markdown-based documentation including a requirements.txt for one-command environment setup (pip install -r requirements.txt).
